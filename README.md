## âš™ï¸ Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Awais748/netflix-clone.git
   cd netflix-clone
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your TMDB and Firebase credentials (refer to `.env.example`).

4. **Start the development server**:
   ```bash
   npm run dev
   ```
